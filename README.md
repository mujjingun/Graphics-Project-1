## Shooting game

### Controls

WASD or arrow keys: Move around

Space bar: Shoot missiles

### Types of objects

* House: 10 Health
* Car: 5 Health
* Cocktail: 2 Health
* Squid: 30 Health
* Balloon : 1 Health, Recovers player's health

![screenshot](screenshot.png)

## Build instructions

### Linux

Use `cmake`.

### Windows

Use `build/graphics1/graphics1.sln`

Only tested in MSVC 2017 x64 Release mode.
